# 📊 Customer Behavior Analysis Project

## 📌 Overview

This project focuses on analyzing customer behavior to uncover patterns, trends, and insights that can help improve business decisions. It demonstrates an end-to-end data analysis workflow, including data cleaning, exploratory analysis, SQL querying, dashboard creation, Reporting and Presentation.

The objective is to understand how customers interact, purchase, and behave, and to translate this data into actionable insights.

---

## 📂 Dataset

* The dataset contains customer-related information such as demographics, purchasing behavior, and transaction details
* Includes both numerical and categorical features
* Used to analyze customer trends, preferences, and patterns

---

## 🛠️ Tools & Technologies

* **Python** – Data loading, cleaning, and EDA
* **PostgreSQL** – Writing and executing SQL queries
* **Power BI** – Building interactive dashboards
* **Gamma** – Creating project presentation (PPT)

---

## 🔄 Project Workflow

### 1. Data Loading

* Loaded dataset using Python (Pandas)
* Inspected structure, columns, and data types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized data formats
* Managed outliers where necessary

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer purchase patterns
* Identified trends and correlations
* Used visualizations to highlight insights

### 4. SQL Analysis (PostgreSQL)

* Imported cleaned data into PostgreSQL
* Wrote SQL queries for business questions
* Performed aggregations, joins, and filtering

### 5. Dashboard Creation (Power BI)

* Designed interactive dashboards
* Visualized KPIs such as customer segments, sales trends, and behavior patterns
* Enabled filtering for better analysis

### 6. Reporting & Presentation

* Created a structured project report
* Developed a presentation using Gamma to summarize insights

---

## 📊 Dashboard Highlights

* Customer segmentation analysis
* Purchase trends over time
* Top-performing categories/products
* Behavior patterns based on demographics
* Interactive filters and visuals

---

## 📈 Results & Insights

* Identified key customer segments
* Found patterns in purchasing behavior
* Highlighted factors influencing customer decisions
* Provided insights to improve marketing and business strategies


## 🚀 Conclusion

This project demonstrates practical skills in data analysis, SQL, and data visualization. It highlights the ability to transform raw customer data into meaningful insights that support data-driven decision-making.
